O que é o protocolo HTTP?

O HTTP (HyperText Transfer Protocol) é um protocolo usado para comunicação na web. Ele permite que navegadores e servidores troquem informações, carregando páginas, imagens e outros recursos.

Como funciona o HTTP?

Quando você acessa um site, seu navegador envia uma requisição HTTP para o servidor. O servidor processa essa requisição e devolve uma resposta com o conteúdo solicitado.

Fluxo de comunicação HTTP

No fluxo acima:

  1. O navegador faz uma requisição HTTP ao servidor.
  2. O servidor processa a requisição e busca os dados.
  3. O servidor responde com o conteúdo solicitado.
  4. O navegador exibe a página para o usuário.

Exemplo de requisição HTTP

Quando acessamos um site, pode ocorrer uma requisição como esta:

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Mozilla/5.0
Accept: text/html

Resposta do servidor

O servidor então envia uma resposta:

HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1024

O que são esses campos?

Cada linha tem um significado:

  • Método: Indica a ação da requisição (ex: GET, POST).
  • URL: O endereço do recurso solicitado.
  • Versão do HTTP: Define o protocolo usado (ex: HTTP/1.1).
  • Cabeçalhos: Informações adicionais (ex: tipo de navegador).
  • Corpo: Se houver dados a serem enviados, ficam aqui.
